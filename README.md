What is this?
This is a game downloader and auto-sideloading app for Quest 2, 3, and 3S. While it has a completely different look and feel compared to [Rookie on Quest](https://github.com/LeGeRyChEeSe/rookie-on-quest/tree/main/app), it retains all essential features while removing unnecessary bloat to focus on simple downloading, installing, and browsing.

# Features
Browse a Huge Library: Access the massive database of games handled by Rookie Sideloader, ensuring all game listings stay up to date.

Game Donations: Contribute to the community! If you have a game that isn't in the database (and isn't uncrackable), you can upload it directly to the servers so this app gives back to the ecosystem.

Auto Sideload: When enabled, the app will prompt you to install the APK (manual) and then automatically sideload the OBB folder, completing the installation for you.

Background & Queue System: Download games in the background while you continue to browse. You can add multiple games to a queue; the app will automatically start the next download once the current one finishes.

Buttery Smooth: Optimized specifically for the Quest 2 to eliminate the lag and crashes found in other loaders. Includes full metadata support, images, clickable game cards for notes, and background syncing.

Faster Downloads & Extraction: Uses a native rclone binary for direct server connections (faster than standard HTTP) and a native 7z binary for lightning-fast extraction with minimal CPU usage.

Local Cache: Metadata is saved locally after the first launch for near-instant startup. The app performs a quick background sync every time it opens to check for new updates and releases.
