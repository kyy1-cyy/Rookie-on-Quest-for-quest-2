Follow these steps to get the app running on your headset:

Get your VR ready: Ensure your Quest 2, 3, or 3S is powered on and developer mode is enabled.

Download the latest release: Go to the Releases page and download the latest .apk file.

Sideload the app: Use any file explorer Mobile VR Station, Cx file explorer (recommended) to install the APK onto your headset.

Grant Permissions: Once you open the app, accept all file permissions.
**Note: This is required to allow the app to write game files and OBBs to your disk, exactly how Rookie On Quest does it.**

Finalize the Setup: Quit the app once the initial setup is complete, then relaunch it.

Browse and Play: You are all set! You can now browse the library, manage your queue, and start sideloading games directly from your headset.
